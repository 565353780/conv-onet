# Conv ONet

## Download

### Dataset

download demo data for test only

```bash
bash ./scripts/download_demo_data.sh
```

or download whole dataset

```bash
bash ./scripts/download_data.sh
```

### Model

```bash
bash ./scripts/download_model.sh
```

## Install

```bash
conda create -n conv python=3.8
conda activate conv
./setup.sh
```

## Run

```bash
python demo.py
```

## Enjoy it~
